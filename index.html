Coding Convention
  • Naming Rules
    • Variables
      • snake_case
        • var_a, var_b, var_c, ...
    • Methods, Classes
      • PascalCase
        • RotateXY(), DrawPipe(), ...
    • Constants
      • UPPER_CASE
        • PI = 3.14 PI = 3.14
  • boolean-specific-prefix
    • use 'is_' prefix
      • is_alive = true
      • is_human = true
      • is_animal = false
